# CHRONIX

CHRONIX is a modern productivity and focus app designed to help users stay consistent, organized, and distraction-free. Built with a clean interface and smooth animations, CHRONIX creates an enjoyable focus experience for studying, working, coding, content creation, and daily productivity.

---

## Features

* Focus & Pomodoro Timer
* Smooth Timer Animations
* Customizable Focus Sessions
* Minimal & Modern UI
* Lightweight and Fast
* Productivity-Friendly Design

---

## Purpose

CHRONIX was created to make productivity simple and motivating. Instead of a cluttered interface, the app focuses on a clean design and smooth user experience that helps users concentrate better and manage time efficiently.

---

## Technologies Used

* HTML
* CSS
* JavaScript

---

## Installation

1. Download or clone the project
2. Open the project folder
3. Run the `index.html` file in your browser
